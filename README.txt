MINIEMPRENDE — OBLEAS (PWA)
--------------------------------
Incluye:
- index.html (app)
- manifest.webmanifest (PWA)
- sw.js (offline)
- icons/ (192, 512 y apple-touch 180)

Cómo usar:
1) Abre index.html (doble clic) para probar.
2) Sube la carpeta a GitHub Pages para instalar en iPhone (Share → Add to Home Screen).
3) Si no ves cambios: edita CACHE en sw.js (v2, v3…).
